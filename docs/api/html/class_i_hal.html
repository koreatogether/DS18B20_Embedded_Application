상세한 설명

Hardware Abstraction Layer (HAL) 인터페이스.

Arduino 프레임워크와 같이 하드웨어에 직접 의존하는 함수들을 추상화합니다. 이를 통해 다른 클래스들이 하드웨어에 직접 의존하지 않고, 이 인터페이스에만 의존하도록 하여 테스트 용이성을 높입니다.
